Company Package Info

Companies attending our Tech Fair will be provided with a table, chairs, bulletin board, and a 10ft by 6ft space to set up in. As before, Tech Fair will run from 10:00am to 4:00pm on October 25th September 27, 28. Lunch will be provided to all employees running the booths.


In order to support smaller companies, Tech Fair's attendance fee depends on the number of employees:

  • $400.00 CAD for companies with under 50 employees
  • $750.00 CAD for companies with over 50 employees
